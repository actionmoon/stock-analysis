# Stock Sell Skill（卖出纪律技能）

> 全局红线（仓位、止损公式、事件分级、优先级仲裁）见 [stock-analyst.md](../../agents/stock-analyst.md)。
> **本 Skill 是 Example 2 铁律的权威源**：任何其他 Skill 引用 `@sell:Example2` 时，以本文件 §四 为准。

## 一、何时调用

- 用户已有持仓，询问"现在该卖吗？""要不要止盈？""要不要止损？"
- 用户出现**情绪化信号**："恐高想落袋""想 T 出来再买回""主观目标价已到"
- 标的出现技术性破位 / 量价背离 / 板块退潮
- 触发 S/A 级突发事件，需紧急减仓或清仓

## 二、4 类卖出触发器

> 优先级：A > C > B > D（参见 AGENT §五）

| 类型 | 触发条件 | 处置 | 是否受 §四 抑制 |
|------|----------|------|-----------------|
| **A. 止损出场** | 跌破技术止损位 / 比例亏损 5%–8% / 时间止损到期 | **无条件清仓或减仓**，不允许犹豫 | **否**（止损优先于一切铁律） |
| **B. 止盈减仓** | 同时出现 ≥ 2 项右侧顶部信号（§三） | 分批减仓（先减 1/3 → 再减 1/3 → 全清） | **是**（@sell:Example2 抑制） |
| **C. 风险事件出场** | S/A 级突发事件、黑名单事件 | 立即清仓或 24h 内调仓 | **否** |
| **D. 调仓置换** | 板块退潮 / 出现更优标的 / 仓位再平衡 | 主动减仓换股，遵守 20%/40%/10% 约束 | **是**（先确认非铁律抑制场景） |

## 三、5 项右侧顶部信号（B 类触发器使用）

> **必须同时出现 ≥ 2 项**才考虑减仓 / 止盈。

1. **放量滞涨**：成交量大幅放大但价格几乎不涨或收阴线（顶部博弈）
2. **放量长上影**：盘中冲高大幅回落，留下长上影（出货特征）
3. **量价背离**：价格创新高但成交量明显萎缩（动能衰竭）
4. **跌破关键均线**：跌破 5 日 / 10 日 / 20 日均线（趋势转弱）
5. **板块或大盘走坏**：所属板块退潮 或 大盘跌破 60 日均线（系统性风险）

### 减仓节奏

| 满足项数 | 动作 | 剩余仓位止损上移到 |
|----------|------|--------------------|
| 2 项 | 减 1/3 | 10 日均线 |
| 3 项 | 再减 1/3 | 5 日均线 |
| 4 项 或 触发硬止损 | 全部清仓 | — |

## 四、铁律 @sell:Example2：放量上涨过程中，不要轻易卖出止盈

**规则**：放量上涨过程中，不要轻易卖出止盈。

**作用范围**：
- 抑制 **B 类（止盈减仓）** 与 **D 类（主观调仓置换）** 的错误冲动
- **不抑制** A 类（止损）与 C 类（风险事件），止损永远优先

**解读**：
- "放量上涨"的标准量价语言含义是**主力建仓 / 趋势确立 / 增量资金入场**，是趋势延续或加速的强信号
- 此阶段任何"恐高减仓 / 见好就收"的行为均违反"让利润奔跑、趋势为王"的纪律
- 必须等待**放量滞涨、放量长上影、量价背离、跌破关键均线**等右侧见顶/转弱信号出现后再卖出

### 4.1 判定条件（满足任一即触发"暂缓止盈"）

1. 当日成交量 ≥ 近 5 日均量的 **1.5 倍**，且收盘较前一日上涨 ≥ **3%**
2. 连续 2 日放量阳线，**站上 / 突破关键阻力位**（前高 / 颈线 / 平台箱体上沿）
3. 个股出现"放量涨停"或"放量长下影 + 收阳"组合形态
4. 板块 + 个股同步放量，且大盘配合（板块强于大盘 + 大盘趋势向上）

### 4.2 正确处置

- **继续持有**，让盈利奔跑
- 上移止损位至**前低 / 5 日均线 / 10 日均线**作为移动止盈
- 浮盈状态下**允许金字塔加仓**（受单票 20% 上限约束，参见 stock-buy §三）
- 仅当出现"右侧顶部信号 ≥ 2 项"才启动减仓

### 4.3 错误处置（明令禁止）

- ❌ "涨得多了，先卖一半锁定利润" — 违反"趋势为王、让利润奔跑"
- ❌ "短线已经赚了 X%，落袋为安" — 主观目标价代替信号判断
- ❌ "估值已经高了，提前下车" — 估值在主升浪中**永远滞后于价格信号**
- ❌ 用"日内 T+0 高抛低吸"反复操作主升浪 — 极易踏空且增加成本

### 4.4 多空对立思考

- 反方逻辑："放量上涨可能是主力对倒拉高出货"——需配合**滞涨 / 长上影 / 量价背离**等右侧确认才成立，而**不是**在放量上涨当日恐慌卖出
- 即使最终是"最后一涨"，等右侧顶部信号再卖只少赚 3%–5% 尖顶利润，但避免在主升浪中段过早下车踏空 30%–50% 尾部利润
- **主升浪只走一次**：宁愿被假突破套住，也不要被恐高情绪甩下车

## 五、心理纪律自检（卖出前必填）

> 防止"情绪化卖出"。任一项答"是"必须重新评估或撤销操作。

| # | 自检题 | 命中后处置 |
|---|--------|-----------|
| 1 | 我卖出的理由是"恐高 / 落袋为安 / 估值贵了"等**主观感受**吗？ | 撤销，回到 §三 看是否有右侧信号 |
| 2 | 我是否因为单日大涨/大跌**急于行动**，而非依据信号？ | 等待至少 1 个交易日再决策 |
| 3 | 我是否因为"前几天没卖错过高点"而**报复性卖出**？ | 撤销，回顾 AGENT §八 复盘机制 |
| 4 | 我是否打算"先卖再低买回来"做 T？ | 不允许（违反趋势为王） |
| 5 | 我是否在 §四 触发期内强行止盈？ | 撤销 |

## 六、清仓冷静期（Cooling-off Period）

完整清仓后立即触发**冷静期**，避免反向追涨被套：

- **24 小时禁买**：清仓后 24h 内不得对**同一标的**重新建仓
- **3 个交易日观察**：观察是否出现"放量企稳 + 右侧确认"
- **再入场规则**：若再次买入，必须重新走完 stock-buy §二 5 重确认 + §三 加仓量化条件，不允许"以原成本回补"

> 冷静期不适用于换股（D 类置换至**不同标的**）。

## 七、卖出决策树

```
是否触发 A 类硬止损？（技术破位 / 比例 5%–8% / 时间到期）
├─ 是 → ✅ 立即清仓或减仓（@sell:Example2 不能阻止 A 类）
└─ 否 → 是否触发 C 类（S/A 级事件 + 黑名单）？
        ├─ 是 → ✅ 立即清仓或 24h 调仓
        └─ 否 → 当前是否处于放量上涨（§四）？
                ├─ 是 → ❌ 暂缓止盈，上移止损位
                └─ 否 → 出现 ≥ 2 项右侧顶部信号？
                        ├─ 是 → 心理纪律自检（§五）通过？
                        │       ├─ 通过 → ✅ 分批减仓
                        │       └─ 不通过 → 撤销操作
                        └─ 否 → ⏸ 持有观察
```

## 九、与其他 Skill 的衔接

- 上游：`stock-analysis` 提供综合评级与止盈/止损位参考
- 上游：`stock-buy` 在建仓时设定的止损位作为本 Skill 的硬约束输入
- 下游：清仓后若仍看好板块 → 重新进入 `stock-selection` 寻找替代标的
- **优先级冲突**：止损（A/C）> @sell:Example2 > B 类止盈 > D 类调仓 > 主观判断

## 十、禁止行为

- ❌ 重述 AGENT 已有红线（仓位、止损、事件分级只引用不抄写）
- ❌ 用 "短线博弈" 绕过冷静期
- ❌ 在 §四 触发期对**未跌破止损**的标的强行止盈
- ❌ 跳过心理纪律自检直接给出卖出价

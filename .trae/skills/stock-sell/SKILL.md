# Stock Sell Skill（卖出纪律技能）

## 何时调用

- 用户已有持仓，询问"现在该卖吗？""要不要止盈？""要不要止损？"；
- 用户出现"恐高想落袋""想 T 出来再买回""主观目标价已到"等情绪信号；
- 标的出现技术性破位 / 量价背离 / 板块退潮；
- 触发 S/A 级突发事件，需要紧急减仓或清仓。

## 核心理念（与 Agent 红线一致）

- **趋势为王，让利润奔跑**：右侧见顶信号确认前不轻易止盈；
- **风险控制优先**：触发止损纪律时**毫不犹豫**执行，不允许"再等等"；
- **不预测顶部，只识别信号**：以放量滞涨 / 长上影 / 量价背离 / 跌破均线为准；
- **多空对立思考**：每个卖出决策都要问"如果是主力洗盘怎么办？"。

---

## 一、卖出触发器分类

| 类型 | 触发条件 | 处置 |
|------|----------|------|
| **A. 止损出场** | 跌破技术止损位 / 比例亏损 5%–8% / 时间止损到期 | **无条件清仓或减仓**，不允许犹豫 |
| **B. 止盈减仓** | 同时出现 ≥ 2 项右侧顶部信号 | 分批减仓（先减 1/3 → 再减 1/3 → 全清） |
| **C. 风险事件出场** | S/A 级突发事件、黑名单事件 | 立即清仓或 24h 内调仓 |
| **D. 调仓置换** | 板块退潮 / 出现更优标的 / 仓位再平衡 | 主动减仓换股，遵守 20%/40%/10% 约束 |

---

## 二、右侧顶部信号清单（用于止盈判断）

> **必须同时出现 ≥ 2 项**才考虑减仓 / 止盈。

1. **放量滞涨**：成交量大幅放大但价格几乎不涨或收阴线（顶部博弈）；
2. **放量长上影**：盘中冲高大幅回落，留下长上影（出货特征）；
3. **量价背离**：价格创新高但成交量明显萎缩（动能衰竭）；
4. **跌破关键均线**：跌破 5 日 / 10 日 / 20 日均线（趋势转弱）；
5. **板块或大盘走坏**：所属板块退潮 或 大盘跌破 60 日均线（系统性风险）。

### 减仓节奏建议

- 出现 2 项 → 减 1/3，剩余 2/3 上移止损位至 10 日均线；
- 出现 3 项 → 再减 1/3，剩余 1/3 上移止损位至 5 日均线；
- 出现 4 项 或 触发硬止损 → 全部清仓。

## 三、铁律 Example 2：放量上涨过程中，不要轻易卖出止盈

> 此规则来自 `stock-analysis/SKILL.md` 的 Example 2，是 sell Skill 的**最高优先级铁律**——
> 它的作用是**抑制错误的止盈冲动**，但**不会**抑制止损（A/C 类触发器优先级更高）。

### 判定条件（满足任一即触发"暂缓止盈"）

1. 当日成交量 ≥ 近 5 日均量的 **1.5 倍**，且收盘较前一日上涨 ≥ **3%**；
2. 连续 2 日放量阳线，**站上 / 突破关键阻力位**（前高 / 颈线 / 平台箱体上沿）；
3. 个股出现"放量涨停"或"放量长下影 + 收阳"组合形态；
4. 板块 + 个股同步放量，且大盘配合（板块强于大盘 + 大盘趋势向上）。

### 正确处置

- **继续持有，让盈利奔跑**；
- 上移止损位至**前低 / 5 日均线 / 10 日均线**作为移动止盈；
- 浮盈状态下**允许金字塔加仓**（受 20% 单票上限约束）；
- 仅当出现"应卖出信号 ≥ 2 项"才启动减仓。

### 错误处置（明令禁止）

- ❌ "涨得多了，先卖一半锁定利润"
- ❌ "短线已经赚了 X%，落袋为安"
- ❌ "估值已经高了，提前下车"
- ❌ 用"日内 T+0 高抛低吸"反复操作主升浪

### 多空对立思考

- 反方逻辑："放量上涨可能是主力对倒拉高出货"——需配合**滞涨 / 长上影 / 量价背离**等右侧确认才成立；
- 即使最终是"最后一涨"，等右侧顶部信号再卖只少赚 3%–5% 尖顶利润，但避免在主升浪中段过早下车踏空 30%–50% 尾部利润；
- **主升浪只走一次**：宁愿被假突破套住，也不要被恐高情绪甩下车。

## 四、卖出决策树

```
是否触发硬止损？（技术破位 / 比例 5%–8% / 时间到期）
├─ 是 → ✅ 立即清仓或减仓（A 类，铁律 Example 2 不能阻止此类）
└─ 否 → 是否有 S/A 级突发事件？
        ├─ 是 → ✅ 立即清仓或 24h 调仓（C 类）
        └─ 否 → 当前是否处于放量上涨？
                ├─ 是 → ❌ 暂缓止盈（Example 2 铁律），上移止损位
                └─ 否 → 是否出现 ≥ 2 项右侧顶部信号？
                        ├─ 是 → ✅ 分批减仓（B 类）
                        └─ 否 → 持有观察
```

## 六、与其他 Skill 的衔接

- 上游：`stock-analysis` 提供综合评级与止盈/止损位参考；
- 上游：`stock-buy` 在建仓时已设定的止损位作为本 Skill 的硬约束；
- 下游：清仓后若仍看好板块 → 重新进入 `stock-selection` 寻找替代标的；
- **优先级冲突时**：止损（A/C）> Example 2 铁律 > 主观判断。

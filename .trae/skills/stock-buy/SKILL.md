# Stock Buy Skill（买入纪律技能）

## 何时调用

- 用户已经完成或参考了 `stock-analysis` 的研判，准备**开仓 / 加仓 / 补仓**；
- 用户询问"现在能买吗？""买点在哪？""能不能抄底？"；
- 用户报告标的已下跌一定幅度，希望"摊低成本"；
- 用户要求计算具体买入价位、止损位与仓位。

## 核心理念（与 Agent 红线一致）

- **不预测，只应对**：右侧买点优先，禁止左侧抄底；
- **金字塔加仓**：首仓 1/3 → 突破确认加 1/3 → 趋势确立加 1/3，**只在盈利时加仓**；
- **量价为王**：买入信号必须有量能配合；
- **黑名单一票否决**：财务造假 / 商誉爆雷 / ST / 违规减持的标的，**不分析、不买入**。

---

## 一、买入触发的 5 重确认（缺一不可）

1. **大盘**：处于上升或震荡向上趋势，未跌破 60 日均线；
2. **板块**：所属板块强于大盘（RPS ↑），处于"启动 / 主升"阶段；
3. **个股 K 线**：站稳 20 / 60 日均线，形态健康（突破 / 回踩不破 / 多头排列）；
4. **量价配合**：放量突破 或 缩量回踩支撑反弹，**无量价背离**；
5. **风险预算**：盈亏比 ≥ 2:1，单笔最大亏损 ≤ 总资金 2%。

> 5 条全部满足 → 进入"金字塔首仓"；缺任一项 → 继续观察，不强行入场。

---

## 二、仓位与买点决策树

```
是否首次建仓？
├─ 是 → 首仓 = 计划仓位 × 1/3
│        └─ 设置初始止损（关键支撑位下方 3%–5%）
│
├─ 否（已有底仓）
│   ├─ 当前盈利 > 0 且突破新高 + 放量 → 金字塔加 1/3
│   ├─ 当前盈利 > 0 且趋势确立（站上更高均线）→ 加最后 1/3
│   └─ 当前亏损 → ❌ 严禁补仓（见 Example 1）
│
└─ 突发事件触发？
    ├─ S 级（黑天鹅）→ ❌ 立即停止买入
    ├─ A 级（重大利空）→ 24h 观察，不买入
    └─ C 级（常规事件）→ 不影响既定纪律
```

### 仓位上限

- 单只个股 ≤ 总资金 20%
- 单板块合计 ≤ 总资金 40%
- 始终保留 ≥ 10% 现金

---

## 三、止损位设置（开仓前必须明确）

| 类型 | 设置方法 |
|------|----------|
| 技术止损 | 跌破关键支撑（20 日均线 / 前低 / 颈线）—— **首选** |
| 比例止损 | 单笔亏损达成本 5%–8% |
| 时间止损 | 买入后 N 周（短线 1–2 周 / 中线 4–6 周）未达预期，主动出局 |

> 三类止损"先达到者执行"。**任何"再等等就回来了"都不允许。**

## 四、铁律 Example 1：放量下跌过程中，不要抄底，禁止补仓买入

> 此规则来自 `stock-analysis/SKILL.md` 的 Example 1，是 buy Skill 的**最高优先级铁律**。

### 判定条件（满足任一即触发）

1. 当日成交量 ≥ 近 5 日均量的 **1.5 倍**，且收盘较前一日下跌 ≥ **3%**；
2. 连续 2 日放量阴线，跌破关键支撑位（20 日均线 / 前低 / 颈线）；
3. 个股出现"放量跌停"或"长上影 + 放量下跌"组合形态。

### 正确处置

- 已有持仓：**触发止损纪律**，按"技术 / 比例 / 时间"先达到者执行减仓或清仓；
- 空仓观察：**保持空仓**，移入"观察池"而非"建仓池"；
- 资金管理：**禁止动用任何加仓额度**（包括金字塔的剩余 1/3、2/3）。

### 错误处置（明令禁止）

- ❌ "跌得多了便宜了，先买点试试"
- ❌ "摊低成本"
- ❌ "基本面没变化，跌出黄金坑"
- ❌ 用"日内 T+0 抢反弹"作为变相加仓

### 失效解除条件（同时满足才允许重新评估买点）

1. 量能萎缩至近 5 日均量的 **70%** 以下（缩量企稳）；
2. 出现 ≥ 1 根放量长下影 / 十字星 / 底分型；
3. 大盘与板块同步企稳，未再创新低；
4. 站回关键均线（如 20 日均线）至少 2 个交易日。

### 多空对立思考

- 反方逻辑："放量下跌也可能是最后一跌"——需配合**衰竭性放量 + 次日缩量收阳**双重确认才成立；
- 即使最终是"最后一跌"，等右侧确认再介入只少赚 5%–10%，但避免被腰斩，**期望值始终更优**。

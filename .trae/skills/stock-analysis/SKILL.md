# Stock Analysis Skill（股票分析技能）

> 全局红线（6 大原则、仓位、止损、事件分级、铁律）见 [stock-analyst.md](../../agents/stock-analyst.md)。
> 本 Skill 仅提供**分析方法论与产出模板**，不重述红线条款。

## 一、何时调用

- 用户请求分析某只股票（A股 / 港股 / 美股）
- 用户希望判断当前是否为合适的**买入/卖出时机**
- 用户提供股票代码、行情、财报，请求解读
- 用户询问某事件（地缘、政策、价格、财报）对标的的影响 → **事件驱动专项分析**
- 用户要求把今日结论与昨日/上次结论做对比 → **比对模式**

## 二、输入要求

| 类别 | 必需信息 | 缺失处置 |
|------|----------|----------|
| 标的 | 代码、名称、所属板块/行业 | 主动询问或检索 |
| 周期 | 短线 / 中线 / 长线 | 默认按用户语境推断 |
| 数据 | K线、量、财报、行业新闻 | 通过工具检索补齐 |
| 目的 | 择时 / 估值 / 风险 / 事件解读 | 主动确认 |
| 持仓档案 | 见 AGENT.md §六 | 影响 buy/sell 衔接，建议补齐 |

## 三、三维分析框架

### 3.1 外在因素（系统性环境）

回答"是否该买股"。

| 维度 | 关注点 |
|------|--------|
| 宏观经济 | GDP / CPI / PPI / PMI、货币（LPR、利率、M2、社融）、财政（减税、专项债、基建） |
| 市场环境 | 流动性（北向、两融、新发基金）、情绪（换手率、涨停数、AH 溢价、VIX）、风格（成长/价值、大盘/小盘） |
| 行业政策 | 产业扶持 / 监管 / 补贴退坡、行业景气度（上下游价、订单、产能利用）、国际（汇率、地缘、大宗、加息） |

**输出**：宏观环境评级 = 利好 / 中性 / 利空 + 主要风险点 + 主要催化剂。

### 3.2 内在因素（基本面）

回答"这只股票值不值得买"。

| 维度 | 关键指标 |
|------|----------|
| 公司质地 | 商业模式、护城河、管理层、治理、商誉占比 |
| 盈利能力 | 毛利率、净利率、ROE、ROIC |
| 成长性 | 营收 YoY、净利润 YoY、扣非增长率 |
| 偿债 / 运营 | 资产负债率、流动比率、应收账款周转 |
| 现金流 | 经营现金流 / 净利润、自由现金流 |
| 估值 | PE / PB / PS / PEG / EV/EBITDA、DCF、近 5–10 年估值分位 |

**输出**：基本面评级 = 优秀 / 良好 / 一般 / 较差 + 合理估值区间 + 安全边际。

### 3.3 技术指标（择时）

**自顶向下三层确认**，避免在弱势市场或弱势板块中选股。

#### Layer 1 大盘
- K 线结构：20/60/120 日均线方向与排列、关键位、形态
- 量价：放量上涨（确认）/ 缩量回调（健康）/ 放量滞涨/下跌（顶部警示）/ 地量地价（底部信号）
- 辅助：MA、MACD、KDJ（周线优先）
- **结论**：趋势 [上升/下降/震荡]，量能 [健康/萎缩/异常]，仓位建议 [积极/标配/谨慎]

#### Layer 2 板块
- 相对强度：板块/大盘比值
- 资金流入：行业 ETF 申购、主力净流入排名
- 轮动位置：启动 / 主升 / 高位 / 退潮
- **结论**：阶段 + 资金 → 是否参与

#### Layer 3 个股
- K 线：站稳 20/60 日均线、形态（突破/回踩/多头排列）、信号（早晨之星、红三兵、下影长十字）
- 量价关系（核心）：

  | 价格 | 量 | 含义 |
  |------|----|------|
  | 上涨 | 放量 | 有效突破，可介入 |
  | 上涨 | 缩量 | 警惕背离 |
  | 下跌 | 放量 | 主力出货 → @buy:Example1 |
  | 下跌 | 缩量 | 健康调整，可观察 |
  | 横盘 | 缩量 | 蓄势待发 |
  | 横盘 | 放量 | 资金博弈，准备突破 |

- 辅助：MACD 金叉、RSI 不超买、布林带下轨反弹、筹码集中度
- **结论**：买点 / 卖点 / 止损位

## 四、行业差异化模板

不同行业的关键指标与估值方法差异显著，分析时**必须套用对应模板**。

| 行业 | 核心指标 | 估值偏好 | 特殊关注 |
|------|----------|----------|----------|
| **银行** | 净息差、不良率、拨备覆盖率、资本充足率、ROE | PB | 房地产敞口、地方债敞口、零售/对公结构 |
| **保险** | 内含价值（EV）、新业务价值（NBV）、综合成本率 | PEV | 投资端权益占比、利率敏感度 |
| **券商** | 经纪/投行/自营/资管收入结构、佣金率 | PB / PE | 市场成交量、IPO 节奏 |
| **白酒/消费** | 批价、渠道库存、动销、吨价、毛利率 | PE | 高端/次高端结构、消费税预期 |
| **医药** | 集采影响、研发管线、专利悬崖、医保谈判 | PE / PEG | 创新药 vs 仿制药占比 |
| **半导体/科技** | 库存周期、ASP、产能利用率、研发费用率 | PE / PS | 美国出口管制、国产替代率 |
| **新能源车 / 锂电** | 渗透率、单车 ASP、原材料（碳酸锂）价格 | PE / PEG | 价格战、产能过剩 |
| **光伏** | 多晶硅价格、组件价格、一体化布局 | PE / PB | 海外贸易壁垒、储能配套 |
| **航运 / 油运** | TCE（VLCC/MR/PCTC）、吨海里需求、地缘事件 | PB / EV/EBITDA | 霍尔木兹/红海/巴拿马运河、OPEC+ |
| **航空 / 机场** | 客座率、ASK、油价、汇率、时刻量 | PB / EV/EBITDA | 国际线恢复、免税业务 |
| **周期（钢铁/煤炭/化工）** | 商品价格、库存、产能利用率 | PB | 供给侧、能耗双控 |
| **互联网平台** | MAU / DAU、ARPU、Take Rate | PE / PS | 监管、反垄断、生成式 AI 影响 |

> 套用模板时，**必须先完成行业归类**；横跨多业务的公司按收入/利润占比加权。

## 五、事件驱动专项分析

当用户询问"某事件对标的是否利好/利空"时，按以下结构作答：

```
【事件】<事件描述>
【级别】S / A / B / C（参见 AGENT.md §3.2）
【时间】事件发生时间 + 当前时间
【作用链路】
  1. 事件 → 一级影响（行业/价格/政策）
  2. 一级 → 二级影响（公司收入/成本/估值）
  3. 二级 → 三级影响（K 线/量价/资金流）
【量化评估】
  - 收入/利润弹性：±X%
  - 估值敏感度：PE/PB 上下浮动 X%
  - 持续性：短期 / 中期 / 长期
【结论】利好 / 中性 / 利空（标注置信度：高/中/低）
【操作建议】维持 / 加仓 / 减仓 / 清仓 → 移交对应 buy/sell Skill
【主要风险（≥3 条）】
【失效信号】<事件反转或量化指标偏离时的信号>
```

## 六、跨日比对模式

当用户要求"今日 vs 昨日/上次结论"时：

| 比对维度 | 输出格式 |
|----------|----------|
| 大盘 | 趋势是否变化？量能变化？仓位建议是否需要调整？ |
| 板块 | 阶段是否切换？资金流入是否反转？ |
| 个股 K 线 | 关键位是否突破/跌破？形态是否完成？ |
| 量价 | 是否触发 @buy:Example1 / @sell:Example2 ？ |
| 估值 | PE/PB 分位变化 |
| 事件 | 是否新增 S/A 级事件？ |
| 结论 | 维持 / 微调 / 反转，并说明理由 |

> 比对结论必须明确"前次有效期是否已过"——若未过，新结论应**承接**而非**推翻**前次结论的风险提示。

## 八、与其他 Skill 的衔接

- 上游：`stock-selection` 提供的候选池 → 本 Skill 做深度研判
- 下游：研判倾向 →
  - 买入/加仓 → `stock-buy`（带上止损位、目标价、行业模板结论）
  - 减仓/止盈/止损 → `stock-sell`（带上当前持仓档案）
- 事件驱动 → 按 §五 输出后，按级别（S/A/B/C）分别移交对应 Skill

## 九、禁止行为

- ❌ 重复 AGENT.md 已有的红线条款（直接引用）
- ❌ 在分析中越权出具具体加仓/减仓比例（属于 buy/sell Skill 职责）
- ❌ 对黑名单标的（财务造假 / ST / 商誉爆雷 / 违规减持）出具买入倾向
- ❌ 在数据缺失或过期时给出确定性结论

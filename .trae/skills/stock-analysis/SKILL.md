# 核心原则

本 Skill 在执行任何分析与建议前，必须遵循以下五条核心原则。它们是凌驾于所有分析方法之上的"红线"。

## 1. 时效性（Timeliness）
- **数据新鲜度**：行情、财报、新闻必须使用**最新**可获取数据；超过 24 小时的行情视为过期，需重新抓取。
- **结论有效期**：每份分析报告必须标注"分析时间"与"有效期"（短线 ≤ 3 日，中线 ≤ 2 周，长线 ≤ 1 季度）。
- **滚动校准**：宏观、行业、个股数据更新时，需重新校准结论，避免使用过时判断。
- **避免幸存者偏差**：历史回测结论需注明样本区间，不直接外推到当前。

## 2. 响应：突发事件（Event-Driven Response）
- **事件分级**
  | 等级 | 类型 | 处置 |
  |------|------|------|
  | S | 黑天鹅（停牌、退市、欺诈、战争、监管巨变） | 立即清仓 / 回避 |
  | A | 重大利空/利好（业绩暴雷、并购、政策转向） | 24h 内调整仓位 |
  | B | 行业级事件（补贴、价格战、技术变革） | 重估板块逻辑 |
  | C | 常规事件（季报、调研、评级调整） | 纳入下次分析 |
- **响应流程**：识别事件 → 判断影响范围（系统/板块/个股）→ 评估持续性 → 调整建议（仓位 / 止损 / 估值）。
- **不预测、只应对**：突发事件出现后以**信号确认**为准，避免凭直觉抢跑。

## 3. 仓位管理（Position Sizing）
- **总仓位规则**（按大盘趋势）
  - 上升趋势：60%–90%
  - 震荡市：30%–60%
  - 下降趋势：≤ 30%，必要时空仓
- **单票上限**：单只个股仓位 ≤ 总资金的 20%；同行业板块合计 ≤ 40%。
- **金字塔加仓**：首仓 1/3 → 突破确认加 1/3 → 趋势确立加 1/3，**只在盈利时加仓，不补亏损**。
- **现金为王**：永远保留 ≥ 10% 现金，应对极端行情与机会捕捉。

## 4. 风险控制（Risk Control）
- **止损纪律**
  - 技术止损：跌破关键支撑位（如 20 日均线 / 前低）
  - 比例止损：单笔亏损达成本的 5%–8% 立即止损
  - 时间止损：买入后 N 周仍未达预期走势，主动出局
- **风险预算**：单笔交易最大亏损 ≤ 总资金的 2%（"2% 法则"）。
- **盈亏比要求**：开仓前估算盈亏比 ≥ 2 : 1，否则放弃。
- **黑名单机制**：财务造假、商誉爆雷、ST 风险、连续违规减持的标的一律不分析、不推荐。
- **风险提示前置**：每份报告"主要风险"必须列出至少 3 条，且不得隐藏。

## 5. 技术分析（Technical Analysis）
- **先大盘后个股**：大盘趋势走坏时，宁可空仓。
- **基本面定方向，技术面定时点**：两者缺一不可。
- **量价为王**：技术指标中量价关系优先级最高。

## 6. 客观与中立（Objectivity）
- **不预设立场**：先看数据再下结论，禁止"为推荐而推荐"。
- **不预测，只应对**：以信号确认为准，避免左侧抄底。
- **多空对立思考**：每个买入逻辑必须配套至少一个**反方观点**与对应的失效信号。

# 实战规则示例（Examples）

以下示例为**强制性铁律**，在分析、建议与操作时必须严格遵循，优先级高于任何"看似便宜"的诱惑。

## Example 1：放量下跌过程中，不要抄底，禁止补仓买入

**规则**：放量下跌过程中，不要抄底，禁止补仓买入。

**解读**：
- "放量下跌"的标准量价语言含义是**主力出货 / 恐慌性抛售**，是趋势转弱或继续下行的强信号；
- 此阶段**任何抄底/补仓行为均违反"不补亏损、不左侧接刀"的纪律**；
- 必须等待**缩量企稳 + 右侧确认信号**（如放量长下影、底分型、突破短期均线）后再行动。

**判定条件（满足任一即触发该规则）**：
1. 当日成交量 ≥ 近 5 日均量的 **1.5 倍**，且收盘价较前一交易日下跌 ≥ **3%**；
2. 连续 2 日放量阴线，跌破关键支撑位（20 日均线 / 前低 / 颈线）；
3. 个股出现"放量跌停"或"长上影 + 放量下跌"组合形态。

**正确处置**：
- 已有持仓：**触发止损纪律**，按"技术止损 / 比例止损 / 时间止损"中先达到者执行减仓或清仓；
- 空仓观察：**保持空仓**，将该标的移入"观察池"而非"建仓池"；
- 资金管理：**禁止动用任何加仓额度**（包括金字塔加仓的剩余 1/3、2/3 仓位）。

**错误处置（明令禁止）**：
- ❌ "跌得多了便宜了，先买点试试" — 违反"不预测、只应对"；
- ❌ "摊低成本" — 违反"只在盈利时加仓，不补亏损"；
- ❌ "基本面没变化，跌出黄金坑" — 量价为王，价格已用脚投票；基本面在放量下跌中**永远滞后于市场信号**；
- ❌ 用"日内 T+0 抢反弹"作为变相加仓 — 同样违反纪律。

**失效解除条件（同时满足才允许重新评估买点）**：
1. 量能萎缩至近 5 日均量的 **70%** 以下（缩量企稳）；
2. 出现至少 1 根**放量长下影/十字星/底分型**；
3. 大盘与板块同步企稳，未再创新低；
4. 站回关键均线（如 20 日均线）至少 2 个交易日。

**多空对立思考**：
- 反方逻辑："放量下跌也可能是最后一跌，地量后必有地价"——成立的前提是出现**衰竭性放量 + 次日缩量收阳**双重确认，而**不是**在放量下跌当日抄底；
- 即使最终证明是"最后一跌"，**等右侧确认再介入也只是少赚 5%–10%**，但避免了被腰斩的尾部风险，**期望值上始终更优**。

## Example 2：放量上涨过程中，不要轻易卖出止盈

**规则**：放量上涨过程中，不要轻易卖出止盈。

**解读**：
- "放量上涨"的标准量价语言含义是**主力建仓 / 趋势确立 / 增量资金入场**，是趋势延续或加速的强信号；
- 此阶段**任何"恐高减仓 / 见好就收"的行为均违反"让利润奔跑、趋势为王"的纪律**；
- 必须等待**放量滞涨、放量长上影、量价背离、跌破关键均线**等右侧见顶/转弱信号出现后再卖出。

**判定条件（满足任一即触发该规则）**：
1. 当日成交量 ≥ 近 5 日均量的 **1.5 倍**，且收盘价较前一交易日上涨 ≥ **3%**；
2. 连续 2 日放量阳线，**站上 / 突破关键阻力位**（前高 / 颈线 / 平台箱体上沿）；
3. 个股出现"放量涨停"或"放量长下影 + 收阳"组合形态；
4. 板块 + 个股 同步放量，且大盘配合（板块强于大盘 + 大盘趋势向上）。

**正确处置**：
- 已有持仓：**继续持有，让盈利奔跑**；可上移止损位至**前低 / 5 日均线 / 10 日均线**作为移动止盈；
- 浮盈状态：**允许金字塔加仓**（首仓 1/3 → 突破确认加 1/3 → 趋势确立加 1/3），符合"只在盈利时加仓"的纪律；
- 仓位管理：可上调单票仓位至上限（≤ 20%），但仍需遵守板块合计 ≤ 40% 的约束。

**错误处置（明令禁止）**：
- ❌ "涨得多了，先卖一半锁定利润" — 违反"趋势为王、让利润奔跑"；
- ❌ "短线已经赚了 X%，落袋为安" — 主观目标价代替信号判断，违反"不预测、只应对"；
- ❌ "估值已经高了，提前下车" — 估值在主升浪中**永远滞后于价格信号**；放量上涨说明市场愿意支付溢价；
- ❌ 用"日内 T+0 高抛低吸"反复操作主升浪 — 极易踏空且增加交易成本。

**应卖出信号（同时出现 ≥ 2 项才考虑减仓 / 止盈）**：
1. **放量滞涨**：成交量大幅放大但价格几乎不涨或收阴线（顶部博弈信号）；
2. **放量长上影**：盘中冲高后大幅回落，留下长上影线（出货特征）；
3. **量价背离**：价格创新高但成交量明显萎缩（动能衰竭）；
4. **跌破关键均线**：跌破 5 日 / 10 日 / 20 日均线（趋势转弱）；
5. **板块或大盘同步走坏**：系统性风险显现，需主动降低敞口。

**多空对立思考**：
- 反方逻辑："放量上涨可能是主力对倒拉高出货"——但需配合**滞涨 / 长上影 / 量价背离**等右侧确认信号才能成立，**而不是**在放量上涨当日恐慌卖出；
- 即使最终证明是"最后一涨"，**等右侧顶部信号再卖出也只是少赚 3%–5% 的尖顶利润**，但避免了在主升浪中段过早下车踏空 30%–50% 的尾部利润，**期望值上始终更优**；
- 经验法则：**主升浪只走一次，错过就是错过**；放量上涨阶段宁愿"被假突破套住"，也不要"在主升浪中被恐高情绪甩下车"。

# Stock Analysis Skill（股票分析技能）

本 Skill 提供一套结构化、可复用的股票分析框架，覆盖**外在因素 → 内在因素 → 技术指标**三层维度，并按"大盘 → 板块 → 个股"自顶向下确认买入时机。

## 何时调用

- 用户请求分析某只股票（A股 / 港股 / 美股）
- 用户希望判断当前是否为合适的**买入时机**
- 用户需要做基本面 + 技术面综合研判
- 用户提供了股票代码、行情数据或财报，请求解读

## 输入要求

调用此 Skill 前，应尽量收集以下信息（缺失项应主动向用户索取或通过工具检索）：

| 类别 | 必需信息 |
|------|----------|
| 标的 | 股票代码、名称、所属板块/行业 |
| 时间 | 分析周期（短线 / 中线 / 长线） |
| 数据 | 近期K线、成交量、财报、行业新闻 |
| 目标 | 用户目的（择时买入 / 估值判断 / 风险评估） |

---

## 一、外在因素分析（宏观与外部环境）

外在因素决定**系统性风险**与**市场风格**，是判断"是否该买股"的前置条件。

### 1.1 宏观经济
- **GDP / CPI / PPI / PMI**：经济周期定位（复苏 / 过热 / 滞胀 / 衰退）
- **货币政策**：利率（LPR、联邦基金利率）、M2、社融、降准降息预期
- **财政政策**：减税、专项债、基建投资、产业刺激

### 1.2 市场环境
- **流动性**：北向资金、两融余额、新发基金、场外资金
- **市场情绪**：换手率、涨停板数、AH溢价、VIX 恐慌指数
- **风险偏好**：风格切换（成长/价值、大盘/小盘）

### 1.3 行业与政策
- **政策导向**：产业扶持、监管收紧、补贴退坡
- **行业景气度**：上下游价格、订单、产能利用率
- **国际环境**：汇率、地缘政治、大宗商品、海外加息周期

### 1.4 输出
- 给出**宏观环境评级**：利好 / 中性 / 利空
- 标注主要风险点与催化剂

---

## 二、内在因素分析（基本面）

内在因素回答"这只股票**值不值得买**"。

### 2.1 公司质地
- **商业模式**：盈利来源、护城河、竞争格局
- **管理层**：股权结构、高管背景、增减持记录
- **治理**：关联交易、商誉占比、合规历史

### 2.2 财务分析（三表 + 关键指标）
| 维度 | 关键指标 |
|------|----------|
| 盈利能力 | 毛利率、净利率、ROE、ROIC |
| 成长性 | 营收 YoY、净利润 YoY、扣非增长率 |
| 偿债 / 运营 | 资产负债率、流动比率、应收账款周转 |
| 现金流 | 经营现金流 / 净利润、自由现金流 |

### 2.3 估值
- **绝对估值**：DCF、股息折现
- **相对估值**：PE（TTM / 动态）、PB、PS、PEG、EV/EBITDA
- **历史百分位**：当前估值在近 5–10 年的分位数

### 2.4 输出
- **基本面评级**：优秀 / 良好 / 一般 / 较差
- **合理估值区间** + **安全边际**

## 三、技术指标分析（择时买入）

采用**自顶向下**三层确认法，避免在弱势市场或弱势板块中选股。

### 3.1 大盘指标：K线 + 量价

**目标**：判断系统性方向，决定**仓位**。

- **K线结构**
  - 趋势：上升 / 下降 / 震荡（看 20、60、120 日均线方向与排列）
  - 关键位：前高、前低、缺口、平台
  - 形态：头肩、双底、三角形整理、旗形
- **量价关系**
  - 放量上涨 → 趋势确认
  - 缩量回调 → 健康洗盘
  - 放量滞涨 / 放量下跌 → 警惕顶部
  - 地量地价 → 关注底部信号
- **辅助指标**：MA、MACD、KDJ（周线优先）

**结论模板**：
> 大盘处于 [上升 / 下降 / 震荡] 趋势，量能 [健康 / 萎缩 / 异常]，建议 [积极 / 标配 / 谨慎] 仓位。

### 3.2 板块指标：K线 + 量价

**目标**：选出**强势赛道**，提高胜率。

- **相对强度**：板块指数 / 大盘指数比值是否上行
- **资金流入**：行业 ETF 申购、主力净流入排名
- **K线 + 量价**：参考 3.1 的方法用在板块指数
- **轮动位置**：处于启动 / 主升 / 高位 / 退潮哪一阶段

**结论模板**：
> 板块处于 [启动 / 主升 / 高位 / 退潮] 阶段，资金 [流入 / 流出]，[建议 / 不建议] 在该板块内选股。

### 3.3 个股指标：K线 + 量价关系

**目标**：精确**买点**与**止损位**。

- **K线**
  - 趋势确认：站稳 20 / 60 日均线
  - 形态：箱体突破、回踩确认、多头排列
  - K线信号：早晨之星、红三兵、下影长十字
- **量价关系**（核心）
  | 价格 | 成交量 | 含义 |
  |------|--------|------|
  | 上涨 | 放量 | 有效突破，可介入 |
  | 上涨 | 缩量 | 警惕背离，谨慎追高 |
  | 下跌 | 放量 | 主力出货，规避 |
  | 下跌 | 缩量 | 健康调整，可观察 |
  | 横盘 | 缩量 | 蓄势，等待方向 |
  | 横盘 | 放量 | 资金博弈，准备突破 |
- **指标辅助**：MACD 金叉、RSI 不超买、布林带下轨反弹、筹码集中度

**买入信号（需多重确认）**：
1. 大盘趋势向上或企稳
2. 所属板块强于大盘
3. 个股放量突破关键位 / 回踩不破支撑
4. 量价配合健康，无背离
5. 设置明确止损位（通常为支撑位下方 3–5%）

## 四、综合输出模板

完成分析后，按以下结构输出报告：

```
【标的】XXX (代码)
【分析周期】短线 / 中线 / 长线

一、外在因素
  - 宏观：……
  - 行业：……
  - 评级：利好 / 中性 / 利空

二、内在因素
  - 商业模式：……
  - 财务亮点 / 风险：……
  - 估值：PE = X，处于历史 X% 分位
  - 评级：优秀 / 良好 / 一般 / 较差

三、技术面（自顶向下）
  - 大盘：趋势 + 量价 → 仓位建议
  - 板块：阶段 + 资金 → 是否参与
  - 个股：K线 + 量价 → 买点 / 止损位

四、综合结论
  - 操作建议：买入 / 观望 / 回避
  - 买入区间：X.XX – X.XX
  - 止损位：X.XX
  - 目标价 / 持有逻辑：……
  - 主要风险：……
```

# Stock Screening Skill（选股技能）

## 何时调用

- 用户没有给出具体标的，希望"挑几只能买的"；
- 用户希望按主题（如 AI、红利、出海、半导体）筛股；
- 用户要求板块轮动 / 风格切换 / 龙头识别；
- 在 `stock-analysis` 之前作为前置步骤使用。

## 核心理念（与 Agent 红线一致）

- **自顶向下**：大盘趋势走坏时，宁可不选股；
- **强者恒强**：只在强势板块中找强势个股，**不在弱势板块抄底**；
- **黑名单优先**：财务造假、商誉爆雷、ST、连续违规减持的标的**直接剔除**，不参与排序；
- **候选池而非推荐**：本 Skill 输出 3–10 只**候选标的**，最终是否买入需交由 `stock-analysis` + `stock-buy` 二次确认。

## 一、三层筛选漏斗

### Layer 1：大盘判断（决定是否选股）

| 大盘状态 | 选股建议 |
|----------|----------|
| 上升趋势（站上 60 日均线 + 量能健康） | 积极选股，仓位 60%–90% |
| 震荡市 | 精选龙头 + 题材轮动，仓位 30%–60% |
| 下降趋势 | **暂停选股**，仓位 ≤ 30%，仅观察 |

> 大盘下行时输出"暂不建议建仓"而非候选池。

### Layer 2：板块筛选（找强势赛道）

筛选标准（满足 ≥ 3 项进入候选板块）：

1. **相对强度（RPS）**：板块指数 / 大盘指数 比值近 20 日上行；
2. **资金流入**：行业 ETF 近 5 日净申购为正，主力净流入排名前 30%；
3. **政策 / 景气催化**：明确产业政策、订单数据、价格信号；
4. **量价配合**：板块指数放量上行或缩量回踩不破 20 日均线；
5. **轮动位置**：处于"启动 / 主升"，避开"高位 / 退潮"。

### Layer 3：个股筛选（多因子打分）

对候选板块内的个股按以下维度打分（百分制，权重示例）：

| 维度 | 权重 | 关键指标 |
|------|------|----------|
| 基本面质地 | 30% | ROE ≥ 12%、近 3 年净利复合增速 ≥ 15%、毛利率稳定 |
| 估值合理 | 15% | PE / PB 处于近 5 年 30%–70% 分位，PEG ≤ 1.2 |
| 趋势强度 | 25% | 站上 60 日均线、20/60/120 多头排列、近 60 日涨幅强于板块 |
| 量价健康 | 15% | 上涨放量、回调缩量、无量价背离 |
| 资金面 | 10% | 北向 / 主力近 5 日净流入、机构持仓比例稳定或上升 |
| 风险扣分 | 5% | 商誉占净资产 > 30%、应收账款 > 营收 50%、质押比例 > 50%（每项扣 5–10 分） |

**入选阈值**：综合得分 ≥ 70 分；同板块入选不超过 2 只，避免过度集中。

## 二、风格化筛选模板

| 风格 | 关键过滤条件 |
|------|--------------|
| 龙头型 | 行业市占率前 3 + 近 1 年涨幅板块前 20% + 机构持仓上升 |
| 红利型 | 股息率 ≥ 4% + 近 5 年连续分红 + 自由现金流为正 + 资产负债率 ≤ 60% |
| 成长型 | 营收 YoY ≥ 30% + 净利润 YoY ≥ 25% + 研发费用率 ≥ 5% |
| 困境反转 | 行业景气度拐点 + 季度净利环比转正 + 技术面站上 120 日均线 |
| 高股息防御 | 股息率前 20% + Beta < 0.8 + 主营稳定 |

> 选择风格前必须与用户确认其投资周期与风险偏好。

## 四、与其他 Skill 的衔接

- 选出候选池后 → 调用 `stock-analysis` 对 Top 1–3 做深度研判；
- 综合研判通过后 → 调用 `stock-buy` 确认买点与仓位；
- 任何已持仓个股出现"放量下跌 / 板块退潮"信号 → 立即移交 `stock-sell`。
